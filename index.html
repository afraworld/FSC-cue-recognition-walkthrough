Why One Classroom教室を一つに限定する理由
The demo shows only the one classroom the pilot is designed around. Different spaces — stairs, elevators, corridors — carry different hazards and different correct actions, so testing them together in a first pilot would make it hard to tell what is actually working.
本デモは、本パイロットが対象とする教室一つのみを示しています。階段・エレベーター・廊下など異なる場所にはそれぞれ異なる危険と適切な行動があるため、最初のパイロットでこれらを同時に検証すると、何が実際に効果を上げているのか判別が難しくなります。
If this pilot shows the approach is worth pursuing, expanding to other spaces would be a separate, later study — each one confirmed with ISI first.
本パイロットによりこの手法に取り組む価値があると示された場合、他の場所への拡張は別途、後続の研究として扱い、いずれもISIとの事前確認を経て実施します。
